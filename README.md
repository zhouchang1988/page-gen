# Page Gen

为文学作品生成精美的响应式单页 HTML 介绍页面。

## 功能特性

- 🎨 **智能配色** — 根据作品类型自动提取主题色
- 📱 **响应式布局** — 完美适配桌面、平板、手机
- ✨ **交互组件** — 滚动进度、导航高亮、淡入动画
- 📖 **丰富分区** — Hero、人物卡片、时间线、语录等
- 🌐 **国内优化** — Google Fonts 镜像加速

## 支持的作品类型

| 类型 | 配色风格 |
|------|---------|
| 武侠/修仙 | 金色 + 翠绿 |
| 奇幻/玄幻 | 冰蓝 + 深蓝 |
| 历史/权谋 | 金色 + 朱红 |
| 神话/志怪 | 紫色 + 粉青 |
| 科幻/赛博 | 霓虹青 + 紫罗兰 |
| 都市/现代 | 蓝白灰 |

## 使用方式

在对话中触发：

```
帮我做一个《三体》的介绍页面
介绍一下《红楼梦》这部作品
生成《进击的动漫》解析页面
```

## 页面结构

```
Hero 区 (全视口标题)
    ↓
粘性导航 (进度指示)
    ↓
作品简介 → 作者介绍 → 世界观设定
    ↓
主要人物 → 情节章节 → 经典语录
    ↓
作品评析 → 页脚
```

## 组件预览

- **Hero 区** — 标题、副标题、关键数据、行动按钮
- **角色卡片** — 头像、姓名、身份、简介
- **时间线** — 事件节点、时间标注
- **引用卡片** — 经典语录、出处标注

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无外部依赖（除 Google Fonts CDN）
- 语义化 HTML5
- CSS 自定义属性（设计令牌）
- IntersectionObserver API（动画）
